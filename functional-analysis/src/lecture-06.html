第六讲：双对偶空间与外测度

Lecturer: Dr. Casey Rodriguez (YouTube)

1. Hahn-Banach 的应用 (Theorem 50)

回顾上一讲的 Hahn-Banach 定理：子空间上满足 $|u(t)| \le C\|t\|$ 的有界线性泛函可以扩展到整个空间，且保持同样的界。本讲首先给出一个重要应用。

定理 50 — Hahn-Banach 的推论

设 $V$ 为赋范空间。对任意 $v \in V \setminus \{0\}$，存在 $f \in V'$（有界线性泛函）使得

$$\|f\| = 1 \quad \text{且} \quad f(v) = \|v\|.$$
证明

定义线性映射 $u : \mathbb{C}v \to \mathbb{C}$（其中 $\mathbb{C}v$ 为 $v$ 的 span）为

$$u(\lambda v) = \lambda \|v\|.$$

这是良定义的，因为 $\mathbb{C}v$ 中每个元素可唯一表示为 $\lambda v$。验证 $|u(t)| \le \|t\|$：对 $t = \lambda v$，

$$|u(\lambda v)| = |\lambda| \cdot \|v\| = \|\lambda v\| = \|t\|.$$

且 $u(v) = \|v\|$。由 Hahn-Banach 定理，存在 $f \in V'$ 扩展 $u$，使得 $|f(t)| \le \|t\|$ 对所有 $t \in V$ 成立。因此 $f(v) = u(v) = \|v\|$，且 $\|f\| \le 1$。又因为 $f\!\left(\frac{v}{\|v\|}\right) = 1$，故 $\|f\| = 1$。$\square$

2. 双对偶空间 (Double Dual)

定义 51 — 双对偶空间 (Double Dual)

赋范空间 $V$ 的双对偶空间定义为

$$V'' = (V')' = B(V', \mathbb{C}),$$

即 $V'$ 上的所有有界线性泛函构成的空间。

例 52 — 典范嵌入映射

固定 $v \in V$，定义 $T_v : V' \to \mathbb{C}$ 为

$$T_v(v') = v'(v), \qquad \forall\, v' \in V'.$$

则 $T_v \in V''$。验证：

  • 线性性：$T_v(v'_1 + v'_2) = (v'_1 + v'_2)(v) = v'_1(v) + v'_2(v) = T_v(v'_1) + T_v(v'_2)$。
  • 有界性：$|T_v(v')| = |v'(v)| \le \|v\| \cdot \|v'\|$，因此 $\|T_v\| \le \|v\|$，故 $T_v$ 确实是有界线性泛函。
定义 53 — 等距算子 (Isometric Operator)

设 $V, W$ 为赋范空间。有界线性算子 $T \in B(V, W)$ 称为等距的 (isometric)，如果对所有 $v \in V$：

$$\|Tv\| = \|v\|.$$

3. 典范嵌入定理 (Theorem 54)

定理 54 — 典范嵌入是等距的

设 $v \in V$，定义 $T_v(v') = v'(v)$。则映射 $T : V \to V''$，$v \mapsto T_v$ 是等距的，即

$$\|T_v\| = \|v\|, \qquad \forall\, v \in V.$$
证明

在例 52 中已证 $\|T_v\| \le \|v\|$。反方向：若 $v = 0$ 则 $\|T_0\| = 0 = \|0\|$，结论成立。

若 $v \ne 0$，由定理 50，存在 $f \in V'$ 使得 $\|f\| = 1$ 且 $f(v) = \|v\|$。于是

$$\|v\| = f(v) = |f(v)| = |T_v(f)| \le \|T_v\| \cdot \|f\| = \|T_v\|.$$

结合 $\|T_v\| \le \|v\|$ 得 $\|T_v\| = \|v\|$，故 $T$ 是等距的。$\square$

注意：等距算子一定是单射（injective），因为 $\|Tv\| = \|v\|$ 意味着只有零向量可以映为零。因此 $T : V \to V''$ 是一个等距嵌入（isometric embedding），即 $V$ 可以等距地嵌入到 $V''$ 中。

4. 自反空间 (Reflexive Spaces)

定义 55 — 自反空间 (Reflexive Space)

Banach 空间 $V$ 称为自反的 (reflexive)，如果典范嵌入 $v \mapsto T_v$ 是满射（surjective），即 $V = V''$（在典范嵌入的意义下）。

例 56 — 自反与非自反空间
  • 对所有 $1 < p < \infty$，$\ell^p$ 是自反的：$(\ell^p)' = \ell^q$，$(\ell^q)' = \ell^p$（其中 $\frac{1}{p} + \frac{1}{q} = 1$）。
  • $\ell^1$ 不是自反的：$(\ell^1)' = \ell^\infty$，但 $(\ell^\infty)' \ne \ell^1$。
  • $c_0$（收敛到 $0$ 的序列空间）不是自反的：$(c_0)' = \ell^1$，$(\ell^1)' = \ell^\infty \ne c_0$。
阶段性总结：至此，关于一般 Banach 空间的理论讨论告一段落。接下来我们转向 Lebesgue 测度与积分理论，这是构建 $L^p$ 空间的基础。

5. Lebesgue 测度的动机

我们已经在序列空间上讨论了 $\ell^p$ 空间。自然地，我们希望对函数定义类似的 $L^p$ 空间。但 Riemann 积分不够用 — 例如 Dirichlet 函数 $\mathbf{1}_{\mathbb{Q}}$（有理数的指示函数）不是 Riemann 可积的。

考虑 Riemann 可积函数空间 $L^1_R([0,1])$，配备范数

$$\|f\|_1 = \int_0^1 |f(x)|\,dx.$$

（严格来说这只是半范数，因为单点处非零的函数积分为零。）问题在于：这个空间不完备，因此不是 Banach 空间。我们需要更一般的积分来获得完备性。

策略：先在 $\mathbb{R}$ 的子集上定义测度 $m(E)$（"集合的长度"），然后基于测度构建积分。对指示函数 $\mathbf{1}_E$，积分应等于 $m(E)$。

对测度的四个期望

  1. 定义域：$m$ 定义在 $\mathcal{P}(\mathbb{R})$（幂集）上，即能测量任意子集。
  2. 区间长度：若 $I$ 为区间，则 $m(I) = \ell(I)$（区间长度），无论开闭。
  3. 可数可加性：若 $\{E_n\}$ 两两不交，则 $m\!\left(\bigcup_n E_n\right) = \sum_n m(E_n)$。
  4. 平移不变性：$m(x + E) = m(E)$，$\forall\, x \in \mathbb{R}$。
Vitali 构造：不幸的是，不存在同时满足以上四条的函数 $m : \mathcal{P}(\mathbb{R}) \to [0,\infty]$。Vitali 构造给出了反例。解决方案：放弃条件 (1)，只在 $\mathbb{R}$ 的部分子集（Lebesgue 可测集）上定义 $m$，同时保持 (2)、(3)、(4)。可测集的族足够大，满足实际需求。

6. 外测度 (Outer Measure)

Carathéodory 的策略：先定义在所有子集上的外测度 $m^*$，它满足条件 (2)、(4) 和"几乎 (3)"，然后限制到行为良好的子集上得到真正的测度。

定义 58 — 外测度 (Outer Measure)

对任意区间 $I \subset \mathbb{R}$，记 $\ell(I)$ 为其长度。对任意子集 $A \subset \mathbb{R}$，定义

$$m^*(A) = \inf\!\left\{\sum_n \ell(I_n) : \{I_n\} \text{ 为可数族开区间，且 } A \subset \bigcup_n I_n\right\}.$$

即用开区间族覆盖 $A$，取所有覆盖的总长度的下确界。显然 $m^*(A) \ge 0$。

例 59 — 单点集的外测度

$m^*(\{0\}) = 0$。对任意 $\varepsilon > 0$，用开区间 $\left(-\frac{\varepsilon}{2},\, \frac{\varepsilon}{2}\right)$ 覆盖 $\{0\}$，其长度为 $\varepsilon$。故

$$0 \le m^*(\{0\}) \le \varepsilon.$$

令 $\varepsilon \to 0$ 得 $m^*(\{0\}) = 0$。类似地，任何有限集的外测度都为 $0$。

7. 外测度的性质

定理 60 — 可数集的外测度为零

若 $A \subset \mathbb{R}$ 是可数集，则 $m^*(A) = 0$。

特别地，$m^*(\mathbb{Q}) = 0$ — 尽管有理数在 $\mathbb{R}$ 中稠密，但它们的"测度"为零。

证明

设 $A$ 可数无穷（有限情形类似）。将 $A$ 枚举为 $A = \{a_1, a_2, a_3, \dots\}$。取任意 $\varepsilon > 0$，对每个 $n \in \mathbb{N}$，令

$$I_n = \left(a_n - \frac{\varepsilon}{2^{n+1}},\; a_n + \frac{\varepsilon}{2^{n+1}}\right).$$

则 $a_n \in I_n$，故 $A \subset \bigcup_n I_n$。各区间长度之和为

$$\sum_n \ell(I_n) = \sum_n \frac{\varepsilon}{2^n} = \varepsilon.$$

因此 $m^*(A) \le \varepsilon$。令 $\varepsilon \to 0$ 得 $m^*(A) = 0$。$\square$

引理 61 — 单调性 (Monotonicity)

若 $A \subset B$，则 $m^*(A) \le m^*(B)$。

证明

$B$ 的任何覆盖也是 $A$ 的覆盖。因此，对 $A$ 的所有覆盖取下确界只可能更小或相等：

$$m^*(A) = \inf\!\left\{\sum \ell(I_n) : A \subset \bigcup I_n\right\} \le \inf\!\left\{\sum \ell(I_n) : B \subset \bigcup I_n\right\} = m^*(B). \quad\square$$
定理 62 — 可数次可加性 (Countable Subadditivity)

设 $\{A_n\}$ 为 $\mathbb{R}$ 的可数族子集（不必不交）。则

$$m^*\!\left(\bigcup_n A_n\right) \le \sum_n m^*(A_n).$$

这是可加性条件 (3) 的"半边"：不等式方向。

证明

若某个 $m^*(A_n) = \infty$ 或 $\sum_n m^*(A_n) = \infty$，则右端已经为 $\infty$，不等式自动成立。

设所有 $m^*(A_n) < \infty$ 且级数收敛。取 $\varepsilon > 0$，对每个 $n$，选取可数族开区间 $\{I_{nk}\}_{k \in \mathbb{N}}$ 覆盖 $A_n$，使得

$$\sum_{k=1}^{\infty} \ell(I_{nk}) < m^*(A_n) + \frac{\varepsilon}{2^n}.$$

则 $\{I_{nk}\}_{n,k \in \mathbb{N}}$ 是可数族开区间，且

$$\bigcup_n A_n \subset \bigcup_{n,k} I_{nk}.$$

因此

$$m^*\!\left(\bigcup_n A_n\right) \le \sum_{n,k} \ell(I_{nk}) = \sum_n \sum_k \ell(I_{nk}) < \sum_n \left(m^*(A_n) + \frac{\varepsilon}{2^n}\right) = \sum_n m^*(A_n) + \varepsilon.$$

令 $\varepsilon \to 0$ 得 $m^*\!\left(\bigcup_n A_n\right) \le \sum_n m^*(A_n)$。$\square$

证明技巧：这里使用了泛函分析中常见的 "$\varepsilon$-trick"：为了证明 $X \le Y$，等价地证明对任意 $\varepsilon > 0$ 有 $X \le Y + \varepsilon$。对每个 $A_n$ 选覆盖时多留 $\varepsilon/2^n$ 的余量，保证总余量为 $\varepsilon$。

本讲小结

概念要点
Hahn-Banach 应用定理 50：对任意 $v \ne 0$，存在 $f \in V'$ 使 $\|f\|=1$，$f(v)=\|v\|$
双对偶空间 $V''$定义 51：$V'' = (V')'$；典范映射 $T_v(v') = v'(v)$ 给出 $V \hookrightarrow V''$
等距嵌入定理 54：$v \mapsto T_v$ 是等距的，即 $\|T_v\| = \|v\|$
自反空间定义 55：$V$ 自反 $\iff$ $v \mapsto T_v$ 满射；$\ell^p$（$1 < p < \infty$）自反，$\ell^1, c_0$ 不自反
Lebesgue 测度动机Riemann 积分不完备；四条期望性质不能同时满足（Vitali 构造）
外测度 $m^*$定义 58：用开区间覆盖取下确界；可数集外测度为零（定理 60）
外测度性质单调性（引理 61）与可数次可加性（定理 62）：$m^*(\bigcup A_n) \le \sum m^*(A_n)$
