1. 回顾与新的完备性刻画 (Completeness via Summability)

上一讲我们引入了 Banach 空间的定义：完备的赋范空间。判断完备性的标准方法是检验每个 Cauchy 序列是否收敛。本讲给出一个等价刻画，在后续积分理论中更为实用。

定义 17 — 可求和与绝对可求和

设 $\{v_n\}_{n=1}^\infty$ 是赋范空间 $V$ 中的序列。

  • 级数 $\sum_n v_n$ 是可求和的 (summable)，如果部分和序列 $\left\{\sum_{n=1}^m v_n\right\}_{m=1}^\infty$ 在 $V$ 中收敛。
  • 级数 $\sum_n v_n$ 是绝对可求和的 (absolutely summable)，如果 $\left\{\sum_{n=1}^m \|v_n\|\right\}_{m=1}^\infty$ 收敛（即 $\sum_n \|v_n\| < \infty$）。
命题 18

如果 $\sum_n v_n$ 绝对可求和，则部分和序列 $\left\{\sum_{n=1}^m v_n\right\}_{m=1}^\infty$ 是 Cauchy 列。

注意：命题结论是 Cauchy 列，而非收敛。要得到收敛，还需空间完备。这正是下面核心定理的要点。

2. 核心定理：Banach 空间的求和刻画 (Theorem 19)

定理 19 — Banach 空间的等价刻画

赋范空间 $V$ 是 Banach 空间，当且仅当 $V$ 中每个绝对可求和的级数都是可求和的。

证明

($\Rightarrow$) 正向：设 $V$ 是 Banach 空间。若 $\sum_n v_n$ 绝对可求和，则由命题 18，部分和序列是 Cauchy 列。由完备性，Cauchy 列收敛，即级数可求和。

($\Leftarrow$) 反向：设每个绝对可求和级数都可求和，需证 $V$ 完备。取任意 Cauchy 序列 $\{v_n\}$，我们构造一个收敛子列（由此可推出全列收敛）。

构造子列：对每个 $k \in \mathbb{N}$，由 Cauchy 性存在 $N_k$ 使得 $\forall\, n, m \ge N_k$：

$$\|v_n - v_m\| < 2^{-k}$$

（选 $2^{-k}$ 是因为它可求和。）定义 $n_k = N_1 + \cdots + N_k$，则 $n_1 < n_2 < \cdots$ 且 $n_k \ge N_k$。于是：

$$\|v_{n_{k+1}} - v_{n_k}\| < 2^{-k}$$

因此级数

$$\sum_{k \in \mathbb{N}} (v_{n_{k+1}} - v_{n_k})$$

绝对可求和（因为 $\sum_{k} 2^{-k} = 1 < \infty$）。由假设，该级数可求和，即部分和

$$\sum_{k=1}^m (v_{n_{k+1}} - v_{n_k}) = v_{n_{m+1}} - v_{n_1}$$

在 $V$ 中收敛。加上常数 $v_{n_1}$ 不改变收敛性，故子列 $\{v_{n_{m+1}}\}$ 收敛。

最后，Cauchy 序列的收敛子列蕴含全列收敛（标准度量空间论证：三角不等式）。因此 $V$ 完备。$\square$

教授提示：这个刻画在证明积分相关空间完备时往往比直接使用 Cauchy 序列定义更方便。

3. 线性算子 (Linear Operators)

确立了 Banach 空间的理论后，我们转向无穷维空间中的"矩阵"类比 — 算子 (operators) 和泛函 (functionals)。下面是一个核心的启发性例子。

例 20 — 积分算子 (Integral Operator)

设 $K: [0,1] \times [0,1] \to \mathbb{C}$ 连续。对任意 $f \in C([0,1])$，定义

$$Tf(x) = \int_0^1 K(x,y)\, f(y)\, dy$$

则 $T$ 是从 $C([0,1])$ 到 $C([0,1])$ 的映射。可以验证：

  • $Tf$ 仍是连续函数（积分的连续性）
  • $T$ 满足线性：$T(\lambda_1 f_1 + \lambda_2 f_2) = \lambda_1 Tf_1 + \lambda_2 Tf_2$

这类积分算子是微分算子的逆 — 后续课程会详细讨论。函数 $K$ 称为核 (kernel)。

定义 21 — 线性算子 (Linear Operator)

设 $V, W$ 是向量空间。映射 $T: V \to W$ 是线性的，如果对所有 $\lambda_1, \lambda_2 \in \mathbb{K}$，$v_1, v_2 \in V$：

$$T(\lambda_1 v_1 + \lambda_2 v_2) = \lambda_1 Tv_1 + \lambda_2 Tv_2$$
术语：我们常用"线性算子 (linear operator)"而非"线性映射"或"线性变换"，三者含义相同。

4. 连续性与有界性等价 (Theorem 22)

在有限维空间中，所有线性变换都自动连续。但在无穷维 Banach 空间之间的线性映射并非如此 — 连续性变成了需要验证的性质。

定理 22 — 连续 $\iff$ 有界

设 $V, W$ 是赋范空间，$T: V \to W$ 是线性算子。则 $T$ 连续当且仅当存在 $C > 0$ 使得对所有 $v \in V$：

$$\|Tv\|_W \le C\|v\|_V$$

此时称 $T$ 为有界线性算子 (bounded linear operator)。

"有界"的含义：这里"有界"不是说 $T$ 的像有界（满足这个条件的线性映射只有零映射），而是说 $V$ 中的有界子集在 $T$ 下的像也是 $W$ 中的有界子集。
证明

($\Leftarrow$) 有界 $\Rightarrow$ 连续：设 $\exists\, C>0$ 使得 $\|Tv\| \le C\|v\|$。取 $v_n \to v$，则

$$\|Tv_n - Tv\| = \|T(v_n - v)\| \le C\|v_n - v\| \to 0$$

因此 $Tv_n \to Tv$，即 $T$ 连续。

($\Rightarrow$) 连续 $\Rightarrow$ 有界：$T$ 连续，则 $T^{-1}(B_W(0,1))$ 是 $V$ 中的开集。由于 $T(0) = 0 \in B_W(0,1)$，故 $0 \in T^{-1}(B_W(0,1))$。由开集性，存在 $r > 0$ 使得 $B_V(0,r) \subseteq T^{-1}(B_W(0,1))$，即

$$T(B_V(0,r)) \subseteq B_W(0,1)$$

取 $C = \tfrac{2}{r}$。对任意 $v \ne 0$，向量 $\tfrac{r}{2\|v\|}v$ 的范数为 $\tfrac{r}{2} < r$，因此

$$\left\|T\!\left(\frac{r}{2\|v\|}v\right)\right\| < 1$$

利用 $T$ 的线性和范数的齐次性：

$$\frac{r}{2\|v\|}\|Tv\| < 1 \;\;\Longrightarrow\;\; \|Tv\| \le \frac{2}{r}\|v\| = C\|v\| \qquad\square$$

5. 积分算子的有界性 (Example 23)

例 23 — 例 20 中的积分算子是有界的

回到积分算子 $Tf(x) = \int_0^1 K(x,y)f(y)\,dy$，我们在 $C([0,1])$ 上使用 $\|\cdot\|_\infty$ 范数。对任意 $f \in C([0,1])$：

$$|Tf(x)| = \left|\int_0^1 K(x,y)f(y)\,dy\right| \le \int_0^1 |K(x,y)|\,|f(y)|\,dy$$

以 $\|f\|_\infty$ 和 $\|K\|_\infty = \sup_{(x,y)\in[0,1]^2}|K(x,y)|$ 分别控制 $|f(y)|$ 和 $|K(x,y)|$：

$$\le \|K\|_\infty \cdot \|f\|_\infty \cdot \int_0^1 dy = \|K\|_\infty \cdot \|f\|_\infty$$

对所有 $x$ 取 sup：$\|Tf\|_\infty \le \|K\|_\infty \cdot \|f\|_\infty$。因此 $T$ 有界（取 $C = \|K\|_\infty$），从而连续。

6. 算子范数与 $B(V,W)$ (Operator Norm)

定义 24 — 有界线性算子空间

设 $V, W$ 是赋范空间。从 $V$ 到 $W$ 的所有有界线性算子的集合记为 $B(V, W)$。可以验证 $B(V,W)$ 本身是一个向量空间。

定义 25 — 算子范数 (Operator Norm)

对 $T \in B(V, W)$，定义

$$\|T\| = \sup_{\|v\|=1,\, v\in V} \|Tv\|$$

这是一个有限数：由有界性 $\|Tv\| \le C\|v\| = C$（当 $\|v\|=1$），而算子范数恰好是使不等式成立的最小常数 $C$。

由算子范数的定义，对所有 $v \in V$：

$$\|Tv\| \le \|T\| \cdot \|v\|$$
定理 26 — 算子范数确实是范数

$\|\cdot\|$ 是 $B(V,W)$ 上的范数，从而 $B(V,W)$ 是赋范空间。

证明

正定性：$\|T\| = 0$ 意味着 $\|Tv\| = 0$ 对所有 $\|v\|=1$。对任意 $v' \ne 0$，$Tv' = \|v'\| T\!\left(\frac{v'}{\|v'\|}\right) = 0$，故 $T$ 是零算子。

齐次性：

$$\|\lambda T\| = \sup_{\|v\|=1}\|\lambda Tv\| = \sup_{\|v\|=1}|\lambda|\,\|Tv\| = |\lambda|\,\|T\|$$

三角不等式：对 $S, T \in B(V,W)$ 和 $\|v\|=1$：

$$\|(S+T)v\| = \|Sv + Tv\| \le \|Sv\| + \|Tv\| \le \|S\| + \|T\|$$

对所有 $\|v\|=1$ 取 sup：$\|S+T\| \le \|S\| + \|T\|$。$\square$

7. $B(V,W)$ 的完备性 (Theorem 27)

定理 27 — $B(V,W)$ 是 Banach 空间

设 $V$ 是赋范空间，$W$ 是 Banach 空间，则 $B(V,W)$ 关于算子范数是完备的，即 $B(V,W)$ 是 Banach 空间。

证明（使用定理 19 的刻画）

设 $\{T_n\}$ 是 $B(V,W)$ 中的绝对可求和序列：$C = \sum_n \|T_n\| < \infty$。需证 $\sum_n T_n$ 可求和。

Step 1：构造候选算子。对任意 $v \in V$：

$$\sum_{n=1}^m \|T_n v\| \le \sum_{n=1}^m \|T_n\|\,\|v\| \le C\|v\|$$

因此 $\sum_n T_n v$ 在 $W$ 中绝对可求和。由 $W$ 是 Banach 空间，该级数可求和。定义

$$Tv = \lim_{m\to\infty} \sum_{n=1}^m T_n v$$

Step 2：$T$ 是线性的。由每个 $T_n$ 线性，

$$T(\lambda_1 v_1 + \lambda_2 v_2) = \lim_{m\to\infty}\sum_{n=1}^m T_n(\lambda_1 v_1 + \lambda_2 v_2) = \lambda_1 Tv_1 + \lambda_2 Tv_2$$

Step 3：$T$ 有界。对任意 $v \in V$：

$$\|Tv\| = \lim_{m\to\infty}\left\|\sum_{n=1}^m T_n v\right\| \le \lim_{m\to\infty}\sum_{n=1}^m \|T_n\|\,\|v\| = C\|v\|$$

故 $T \in B(V,W)$。

Step 4：算子范数收敛。对 $\|v\|=1$：

$$\left\|Tv - \sum_{n=1}^m T_n v\right\| = \left\|\lim_{m'\to\infty}\sum_{n=m+1}^{m'} T_n v\right\| \le \sum_{n=m+1}^\infty \|T_n\|$$

对所有 $\|v\|=1$ 取 sup：

$$\left\|T - \sum_{n=1}^m T_n\right\| \le \sum_{n=m+1}^\infty \|T_n\| \to 0 \quad (m\to\infty)$$

因为这是收敛级数的尾部。因此 $\sum_n T_n$ 在算子范数下收敛到 $T$。$\square$

证明结构：与证明 $C_b(X)$ 完备的思路非常相似 — 先逐点构造候选，再证候选属于空间，最后证收敛。Blueprint 再次发挥作用。

8. 对偶空间 (Dual Space)

定义 28 — 对偶空间与泛函

设 $V$ 是赋范空间（域 $\mathbb{K}$）。$V$ 的对偶空间 (dual space) 定义为

$$V' = B(V, \mathbb{K})$$

即从 $V$ 到标量域 $\mathbb{K}$ 的所有有界线性算子的空间。$V'$ 的元素称为泛函 (functional)。

由于 $\mathbb{K} = \mathbb{R}$ 或 $\mathbb{C}$ 都是完备的（即 Banach 空间），定理 27 立刻给出：

推论

对任意赋范空间 $V$，其对偶空间 $V'$ 总是 Banach 空间。

$\ell^p$ 空间的对偶

事实 — $(\ell^p)' = \ell^{p'}$

设 $1 \le p \le \infty$，$p'$ 是 $p$ 的共轭指标 (conjugate exponent)，满足

$$\frac{1}{p} + \frac{1}{p'} = 1$$

（约定 $p=1 \Leftrightarrow p'=\infty$）。则：

  • $(\ell^1)' = \ell^\infty$
  • $(\ell^2)' = \ell^2$（$\ell^2$ 是唯一自对偶的 $\ell^p$ 空间）
  • $(\ell^p)' = \ell^{p'}$，对 $1 < p < \infty$
注意：$(\ell^\infty)'$ 不是 $\ell^1$！（如果是的话，生活会简单得多。）$\ell^\infty$ 的对偶空间比 $\ell^1$ 大得多，这个现象在 $L^p$ 空间中同样存在。这是泛函分析中的一个重要"不对称性"。
