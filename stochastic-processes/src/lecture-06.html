第六讲：从 Poisson 过程到 Markov 链

Lecturer: Prof. Robert Gallager / TA: Mina Karzand (MIT OCW)

1. Poisson 过程性质回顾

在前几讲中，我们详细研究了速率为 $\lambda$ 的 Poisson 过程。本讲首先回顾其关键性质，然后从 Poisson 过程的条件密度出发，自然地引出 Markov 链的概念。

核心性质汇总

  • 到达间隔：$X_1, X_2, \ldots$ 是 IID $\text{Exp}(\lambda)$ 随机变量。
  • 计数分布：$N(t) \sim \text{Poisson}(\lambda t)$，即 $\Pr\{N(t) = n\} = \frac{(\lambda t)^n e^{-\lambda t}}{n!}$。
  • 平稳独立增量：不相交时间区间上的增量相互独立，增量分布仅依赖区间长度。
  • 无记忆性：从任意时刻 $t$ 开始，剩余等待时间仍服从 $\text{Exp}(\lambda)$，与过去独立。
  • 小增量条件：$\Pr\{N(t, t+\delta] = 1\} = \lambda\delta + o(\delta)$，$\Pr\{N(t, t+\delta] \geq 2\} = o(\delta)$。
从连续到离散的动机：Poisson 过程描述的是连续时间内事件的随机到达。但在许多实际系统（排队网络、通信协议、库存管理）中，我们更关心系统在每次事件发生时的状态变化——例如队列长度、信道占用数等。这些离散状态的随时间演变构成了 Markov 链理论的基础。

2. Poisson 过程的联合条件密度

在引入 Markov 链之前，我们考察 Poisson 过程到达时刻的条件性质，这揭示了"给定当前信息后未来与过去无关"这一核心思想。

给定 $N(t) = n$ 时到达时刻的条件分布

定理 — 条件均匀性 (Order Statistics Property)

设 $\{N(t);\, t > 0\}$ 是速率为 $\lambda$ 的 Poisson 过程。给定 $N(t) = n$，则 $n$ 个到达时刻 $(S_1, S_2, \ldots, S_n)$ 的条件联合密度与 $n$ 个独立均匀分布 $U(0, t)$ 随机变量的顺序统计量 (order statistics) 具有相同的分布：

$$f_{S_1, \ldots, S_n | N(t)=n}(s_1, \ldots, s_n) = \frac{n!}{t^n}, \quad 0 < s_1 < s_2 < \cdots < s_n < t$$
证明

回忆到达时刻的联合密度为 $f_{\vec{S}^n}(s_1, \ldots, s_n) = \lambda^n e^{-\lambda s_n}$（在 $0 < s_1 < \cdots < s_n$ 上）。但这对应的是 $S_n$ 的边际密度在第 $n$ 次到达后过程继续的情形。

更精确地，我们需要计算：

$$f_{S_1,\ldots,S_n | N(t)=n}(s_1,\ldots,s_n) = \frac{f_{S_1,\ldots,S_n}(s_1,\ldots,s_n) \cdot \Pr\{X_{n+1} > t - s_n\}}{\Pr\{N(t) = n\}}$$

其中分子中 $\Pr\{X_{n+1} > t - s_n\} = e^{-\lambda(t-s_n)}$。因此分子为：

$$\lambda^n e^{-\lambda s_n} \cdot e^{-\lambda(t - s_n)} = \lambda^n e^{-\lambda t}$$

分母为 $\Pr\{N(t) = n\} = \frac{(\lambda t)^n e^{-\lambda t}}{n!}$。两者相除：

$$f_{S_1,\ldots,S_n | N(t)=n} = \frac{\lambda^n e^{-\lambda t}}{(\lambda t)^n e^{-\lambda t}/n!} = \frac{n!}{t^n} \qquad\square$$
含义：给定在 $[0, t]$ 内发生了 $n$ 次到达，这些到达时刻的具体位置就像把 $n$ 个点"随机均匀地撒"在区间 $[0, t]$ 上一样。这一性质在模拟 Poisson 过程、分析网络流量等方面极为有用。

Poisson 过程的 Markov 性质

Poisson 过程的计数过程 $\{N(t)\}$ 满足一个关键性质：

$$\Pr\{N(t+s) - N(t) = k \mid N(\tau),\, 0 \leq \tau \leq t\} = \Pr\{N(t+s) - N(t) = k \mid N(t)\}$$

即给定当前状态 $N(t)$ 后，未来的增量与整个过去历史条件独立。这正是Markov 性质的原型，它激励我们研究一般的 Markov 链。

3. Markov 链的定义

我们现在从连续时间的 Poisson 过程过渡到离散时间的 Markov 链。Markov 链是一类极其重要的随机过程，其核心思想是：系统的未来只依赖当前状态，而与到达当前状态的路径无关。

定义 — 离散时间 Markov 链 (Discrete-Time Markov Chain, DTMC)

设 $\{X_n;\, n \geq 0\}$ 是取值于可数状态空间 $\mathcal{S}$ 的随机过程。若对所有 $n \geq 0$ 和所有状态 $i_0, i_1, \ldots, i_{n-1}, i, j \in \mathcal{S}$，满足

$$\Pr\{X_{n+1} = j \mid X_n = i, X_{n-1} = i_{n-1}, \ldots, X_0 = i_0\} = \Pr\{X_{n+1} = j \mid X_n = i\}$$

则称 $\{X_n\}$ 为 Markov 链。上述条件称为 Markov 性质（Markov property）或无记忆性。

定义 — 时齐 Markov 链 (Time-Homogeneous Markov Chain)

若转移概率不依赖于时间步 $n$，即

$$\Pr\{X_{n+1} = j \mid X_n = i\} = P_{ij} \quad \text{对所有 } n \geq 0$$

则称 Markov 链是时齐的（time-homogeneous）或平稳的（stationary）。此处 $P_{ij}$ 称为从状态 $i$ 到状态 $j$ 的一步转移概率（one-step transition probability）。

注意区分："平稳 Markov 链"指转移概率不随时间变化（时齐），而非指过程具有平稳分布。后者是额外的性质，将在后续讨论。

Markov 性质的等价形式

Markov 性质有几种有用的等价表述：

命题 — Markov 性质的等价形式

以下条件等价：

  1. $\Pr\{X_{n+1} = j \mid X_n = i, X_{n-1} = i_{n-1}, \ldots, X_0 = i_0\} = P_{ij}$
  2. 对任意有界函数 $f$ 和任意 $m \geq 1$：$E[f(X_{n+1}, \ldots, X_{n+m}) \mid X_0, \ldots, X_n] = E[f(X_{n+1}, \ldots, X_{n+m}) \mid X_n]$
  3. 给定 $X_n$ 后，$(X_0, \ldots, X_{n-1})$ 与 $(X_{n+1}, X_{n+2}, \ldots)$ 条件独立。

4. 转移概率矩阵

对于有限状态空间 $\mathcal{S} = \{1, 2, \ldots, M\}$ 的时齐 Markov 链，所有一步转移概率可以组织成一个矩阵。

定义 — 转移概率矩阵 (Transition Probability Matrix)

转移概率矩阵（也称随机矩阵）$\mathbf{P}$ 是一个 $M \times M$ 矩阵，其第 $(i, j)$ 元素为：

$$[\mathbf{P}]_{ij} = P_{ij} = \Pr\{X_{n+1} = j \mid X_n = i\}$$

满足以下条件：

  • $P_{ij} \geq 0$ 对所有 $i, j$（非负性）
  • $\sum_{j \in \mathcal{S}} P_{ij} = 1$ 对所有 $i$（行和为 1）

示例：简单两状态链

考虑状态空间 $\mathcal{S} = \{0, 1\}$ 的 Markov 链，转移矩阵为：

$$\mathbf{P} = \begin{pmatrix} 1 - \alpha & \alpha \\ \beta & 1 - \beta \end{pmatrix}$$

其中 $0 < \alpha, \beta < 1$。这意味着：

  • 在状态 0 时，以概率 $\alpha$ 转移到状态 1，以概率 $1-\alpha$ 留在状态 0。
  • 在状态 1 时，以概率 $\beta$ 转移到状态 0，以概率 $1-\beta$ 留在状态 1。

$n$ 步转移概率

定义 — $n$ 步转移概率

从状态 $i$ 出发，经过 $n$ 步到达状态 $j$ 的概率定义为：

$$P_{ij}^{(n)} = \Pr\{X_{n+m} = j \mid X_m = i\}$$

（由时齐性，上式与 $m$ 无关。）约定 $P_{ij}^{(0)} = \delta_{ij}$（Kronecker delta）。

定理 — $n$ 步转移矩阵等于矩阵幂

$n$ 步转移概率矩阵 $\mathbf{P}^{(n)}$（其第 $(i,j)$ 元素为 $P_{ij}^{(n)}$）满足：

$$\mathbf{P}^{(n)} = \mathbf{P}^n$$

即 $n$ 步转移矩阵是一步转移矩阵的 $n$ 次方。

证明

由全概率公式和 Markov 性质：

$$P_{ij}^{(n)} = \sum_{k \in \mathcal{S}} P_{ik}^{(n-1)} P_{kj} = [\mathbf{P}^{n-1} \cdot \mathbf{P}]_{ij} = [\mathbf{P}^n]_{ij}$$

用数学归纳法即得 $\mathbf{P}^{(n)} = \mathbf{P}^n$。$\square$

5. Chapman-Kolmogorov 方程

Chapman-Kolmogorov 方程是 Markov 链理论中最基本的递推关系，它将长程转移概率分解为短程转移概率的组合。

定理 — Chapman-Kolmogorov 方程

对任意非负整数 $m, n$ 和所有状态 $i, j$：

$$P_{ij}^{(m+n)} = \sum_{k \in \mathcal{S}} P_{ik}^{(m)} \cdot P_{kj}^{(n)}$$

等价的矩阵形式为：$\mathbf{P}^{m+n} = \mathbf{P}^m \cdot \mathbf{P}^n$。

证明

对中间状态 $X_m$ 做全概率展开：

$$P_{ij}^{(m+n)} = \Pr\{X_{m+n} = j \mid X_0 = i\}$$ $$= \sum_{k \in \mathcal{S}} \Pr\{X_{m+n} = j \mid X_m = k, X_0 = i\} \cdot \Pr\{X_m = k \mid X_0 = i\}$$

由 Markov 性质，$\Pr\{X_{m+n} = j \mid X_m = k, X_0 = i\} = \Pr\{X_{m+n} = j \mid X_m = k\} = P_{kj}^{(n)}$。因此：

$$P_{ij}^{(m+n)} = \sum_{k \in \mathcal{S}} P_{ik}^{(m)} \cdot P_{kj}^{(n)} \qquad\square$$
直观理解：从 $i$ 到 $j$ 走 $m+n$ 步，可以拆分为：先从 $i$ 出发走 $m$ 步到达某个中间状态 $k$，再从 $k$ 出发走 $n$ 步到 $j$。对所有可能的中间状态 $k$ 求和即得。这一方程的强大之处在于它允许我们用矩阵乘法高效计算任意步数的转移概率。

状态分布的演化

设 $\boldsymbol{\pi}^{(n)} = (\pi_1^{(n)}, \pi_2^{(n)}, \ldots, \pi_M^{(n)})$ 为时刻 $n$ 的状态分布向量，其中 $\pi_j^{(n)} = \Pr\{X_n = j\}$。则：

$$\boldsymbol{\pi}^{(n)} = \boldsymbol{\pi}^{(0)} \mathbf{P}^n$$

这说明整个 Markov 链的概率行为完全由初始分布 $\boldsymbol{\pi}^{(0)}$ 和转移矩阵 $\mathbf{P}$ 决定。

6. 状态的分类 (Classification of States)

为理解 Markov 链的长期行为，我们需要对状态进行系统分类。

可达性与互通

定义 — 可达 (Accessible) 与互通 (Communicate)

状态 $j$ 从状态 $i$ 可达（accessible），记作 $i \to j$，如果存在 $n \geq 0$ 使得 $P_{ij}^{(n)} > 0$。

若 $i \to j$ 且 $j \to i$，则称 $i$ 与 $j$ 互通（communicate），记作 $i \leftrightarrow j$。

命题 — 互通是等价关系

互通关系 $\leftrightarrow$ 是状态空间 $\mathcal{S}$ 上的等价关系：

  1. 自反性：$i \leftrightarrow i$（取 $n=0$，$P_{ii}^{(0)} = 1$）
  2. 对称性：$i \leftrightarrow j \Rightarrow j \leftrightarrow i$（由定义直接得到）
  3. 传递性：$i \leftrightarrow j$ 且 $j \leftrightarrow k$ $\Rightarrow$ $i \leftrightarrow k$（由 Chapman-Kolmogorov 方程）

因此状态空间可以被划分为若干互通类（communicating classes）。

定义 — 不可约 (Irreducible)

若整个状态空间 $\mathcal{S}$ 只有一个互通类（即任意两个状态都互通），则称该 Markov 链是不可约的（irreducible）。

常返与暂态

定义 — 首次到达时间 (First Passage Time)

从状态 $i$ 出发，首次到达状态 $j$ 的时间定义为：

$$T_j = \min\{n \geq 1 : X_n = j\}$$

特别地，$T_i$ 是从 $i$ 出发首次返回$i$ 的时间（first return time）。定义首次到达概率：

$$f_{ij}^{(n)} = \Pr\{T_j = n \mid X_0 = i\}, \quad f_{ij} = \Pr\{T_j < \infty \mid X_0 = i\} = \sum_{n=1}^{\infty} f_{ij}^{(n)}$$
定义 — 常返态与暂态 (Recurrent and Transient States)

状态 $i$ 称为常返的（recurrent），如果从 $i$ 出发以概率 1 返回 $i$：

$$f_{ii} = \Pr\{T_i < \infty \mid X_0 = i\} = 1$$

状态 $i$ 称为暂态的（transient），如果 $f_{ii} < 1$。

定理 — 常返与暂态的判定准则

设 $V_i = \sum_{n=0}^{\infty} \mathbf{1}\{X_n = i\}$ 为访问状态 $i$ 的总次数。则：

  1. 状态 $i$ 常返 $\Leftrightarrow$ $E[V_i \mid X_0 = i] = \infty$ $\Leftrightarrow$ $\sum_{n=0}^{\infty} P_{ii}^{(n)} = \infty$
  2. 状态 $i$ 暂态 $\Leftrightarrow$ $E[V_i \mid X_0 = i] = \frac{1}{1 - f_{ii}} < \infty$ $\Leftrightarrow$ $\sum_{n=0}^{\infty} P_{ii}^{(n)} < \infty$
证明思路

若状态 $i$ 暂态（$f_{ii} < 1$），则每次返回 $i$ 后，下次不返回的概率为 $1 - f_{ii} > 0$。访问 $i$ 的次数服从几何分布：$\Pr\{V_i = k \mid X_0 = i\} = f_{ii}^{k-1}(1 - f_{ii})$，$k \geq 1$。因此 $E[V_i \mid X_0 = i] = 1/(1 - f_{ii})$。

另一方面，$E[V_i \mid X_0 = i] = \sum_{n=0}^{\infty} \Pr\{X_n = i \mid X_0 = i\} = \sum_{n=0}^{\infty} P_{ii}^{(n)}$。

若 $f_{ii} = 1$（常返），几何分布退化，$V_i = \infty$ a.s.，故 $\sum_{n=0}^{\infty} P_{ii}^{(n)} = \infty$。$\square$

定理 — 互通类的性质

常返性和暂态性是类性质（class properties）：若 $i \leftrightarrow j$，则 $i$ 常返当且仅当 $j$ 常返。

证明

设 $i \leftrightarrow j$，则存在 $m, l \geq 1$ 使得 $P_{ij}^{(m)} > 0$ 且 $P_{ji}^{(l)} > 0$。由 Chapman-Kolmogorov 方程：

$$P_{jj}^{(m+n+l)} \geq P_{ji}^{(l)} \cdot P_{ii}^{(n)} \cdot P_{ij}^{(m)}$$

因此 $\sum_{n} P_{jj}^{(m+n+l)} \geq P_{ji}^{(l)} P_{ij}^{(m)} \sum_{n} P_{ii}^{(n)}$。若 $i$ 常返，则右边发散，故 $j$ 也常返。对称地可证反向。$\square$

正常返与零常返

定义 — 正常返与零常返

设状态 $i$ 为常返态，定义平均回返时间（mean recurrence time）：

$$\mu_i = E[T_i \mid X_0 = i] = \sum_{n=1}^{\infty} n \cdot f_{ii}^{(n)}$$
  • 若 $\mu_i < \infty$，称 $i$ 为正常返（positive recurrent）。
  • 若 $\mu_i = \infty$，称 $i$ 为零常返（null recurrent）。
有限状态空间：对于有限状态 Markov 链，不可能存在零常返态。因此有限不可约 Markov 链的所有状态都是正常返的。零常返只出现在可数无穷状态空间中（例如对称随机游走 $\mathbb{Z}$ 上的每一个状态）。

周期性

定义 — 状态的周期 (Period of a State)

状态 $i$ 的周期定义为：

$$d(i) = \gcd\{n \geq 1 : P_{ii}^{(n)} > 0\}$$

若 $d(i) = 1$，称状态 $i$ 为非周期的（aperiodic）。若 $d(i) > 1$，称状态 $i$ 为周期的（periodic），周期为 $d(i)$。

命题 — 周期性是类性质

若 $i \leftrightarrow j$，则 $d(i) = d(j)$。因此我们可以讨论一个互通类的周期。

定义 — 遍历 (Ergodic)

一个状态既是正常返的又是非周期的，则称它为遍历的（ergodic）。一个不可约 Markov 链的所有状态都是遍历的，则称该链为遍历链（ergodic chain）。

7. M/M/1 排队模型作为动机示例

为了说明 Markov 链如何自然产生于 Poisson 过程驱动的系统，我们考虑经典的 M/M/1 队列。

模型描述

M/M/1 队列的三个 "M" 分别代表：

  • Markovian 到达：顾客按速率为 $\lambda$ 的 Poisson 过程到达。
  • Markovian 服务：服务时间为参数 $\mu$ 的指数分布。
  • 1 个服务器。

设 $Q(t)$ 为时刻 $t$ 系统中的顾客数（含正在服务的）。在每个"事件时刻"（到达或离去），$Q(t)$ 发生跳跃。如果我们在事件时刻观察系统，会发现队列长度构成一个 Markov 链。

嵌入的离散时间 Markov 链

更精确地，考虑连续时间过程 $\{Q(t)\}$。利用指数分布的无记忆性：

  • 当系统处于状态 $n \geq 1$ 时，下一个事件是到达（以速率 $\lambda$）或离去（以速率 $\mu$），两者独立竞争。
  • 下一个事件是到达的概率为 $\frac{\lambda}{\lambda + \mu}$，是离去的概率为 $\frac{\mu}{\lambda + \mu}$。
  • 当系统处于状态 $0$ 时，只能发生到达（以速率 $\lambda$）。

这给出了状态空间 $\mathcal{S} = \{0, 1, 2, \ldots\}$ 上的转移概率：

$$P_{0,1} = 1, \quad P_{n,n+1} = \frac{\lambda}{\lambda+\mu},\quad P_{n,n-1} = \frac{\mu}{\lambda+\mu} \quad (n \geq 1)$$
关键观察：M/M/1 模型将连续时间的 Poisson 到达过程"嵌入"（embed）为离散时间 Markov 链。这种从连续到离散的转换正是本讲"从 Poisson 到 Markov"的核心主题。系统的无记忆性完全来自到达间隔和服务时间的指数分布。

8. 生灭过程 (Birth-Death Processes)

M/M/1 队列是生灭过程（birth-death process）的一个特例。生灭过程是状态空间为 $\{0, 1, 2, \ldots\}$（或其有限子集）的 Markov 链，其中从任何状态只能转移到相邻状态。

定义 — 生灭链 (Birth-Death Chain)

状态空间为 $\mathcal{S} = \{0, 1, 2, \ldots\}$ 的 Markov 链称为生灭链，如果其转移概率满足：

$$P_{n,n+1} = p_n \quad (\text{出生概率}), \quad P_{n,n-1} = q_n \quad (\text{死亡概率}), \quad P_{n,n} = r_n = 1 - p_n - q_n$$

其中 $p_n > 0$（$n \geq 0$），$q_n > 0$（$n \geq 1$），$q_0 = 0$（反射边界条件）。

生灭链的转移矩阵结构

转移矩阵具有三对角（tridiagonal）结构：

$$\mathbf{P} = \begin{pmatrix} r_0 & p_0 & 0 & 0 & \cdots \\ q_1 & r_1 & p_1 & 0 & \cdots \\ 0 & q_2 & r_2 & p_2 & \cdots \\ 0 & 0 & q_3 & r_3 & \cdots \\ \vdots & & & & \ddots \end{pmatrix}$$

M/M/1 作为特例

对 M/M/1 队列，设 $\rho = \lambda/\mu$（交通强度），则：

$$p_n = \frac{\lambda}{\lambda + \mu} = \frac{\rho}{1+\rho}, \quad q_n = \frac{\mu}{\lambda + \mu} = \frac{1}{1+\rho} \quad (n \geq 1)$$ $$p_0 = 1, \quad q_0 = 0, \quad r_n = 0 \quad \text{（纯生灭，无自环）}$$

一般生灭链的常返性判定

定理 — 生灭链中状态 0 的常返性

对于生灭链，状态 0 是常返的当且仅当：

$$\sum_{n=1}^{\infty} \frac{q_1 q_2 \cdots q_n}{p_1 p_2 \cdots p_n} = \infty$$

等价地，定义 $\gamma_n = \frac{p_0 p_1 \cdots p_{n-1}}{q_1 q_2 \cdots q_n}$（$n \geq 1$），则状态 0 常返 $\Leftrightarrow$ $\sum_{n=1}^{\infty} \gamma_n^{-1} = \infty$，即 $\sum_{n=1}^{\infty} \frac{1}{\gamma_n} = \infty$。

证明思路

考虑从状态 $n$ 出发向左到达状态 0 之前先到达状态 $n+1$ 的概率。设 $\alpha_n = \Pr\{\text{到达 } n+1 \text{ 先于 } 0 \mid X_0 = n\}$。通过递推关系和首步分析可以得到 $\alpha_n$ 的表达式，进而判定从任意状态出发是否以概率 1 返回状态 0。级数 $\sum \gamma_n^{-1}$ 的发散性等价于"从右侧漂移回 0 的确定性"。$\square$

M/M/1 的情形：对于 M/M/1 队列，$p_n/q_n = \lambda/\mu = \rho$（常数），所以 $\gamma_n = \rho^n$。当 $\rho < 1$ 时 $\sum \gamma_n^{-1} = \sum \rho^{-n} = \infty$，状态 0 常返（实际上是正常返）；当 $\rho \geq 1$ 时 $\sum \gamma_n^{-1} < \infty$，状态 0 暂态（队列趋于无穷）。这与直觉一致：只有当服务速率超过到达速率时，队列才是稳定的。

9. 平稳分布（初步介绍）

Markov 链长期行为的核心问题之一是：经过足够长的时间后，系统的状态分布是否会收敛到一个固定的分布？这就是平稳分布的概念。

定义 — 平稳分布 (Stationary Distribution)

概率向量 $\boldsymbol{\pi} = (\pi_j)_{j \in \mathcal{S}}$（$\pi_j \geq 0$，$\sum_j \pi_j = 1$）称为 Markov 链的平稳分布（或不变分布、稳态分布），如果：

$$\boldsymbol{\pi} = \boldsymbol{\pi} \mathbf{P}$$

即对所有 $j \in \mathcal{S}$：$\pi_j = \sum_{i \in \mathcal{S}} \pi_i P_{ij}$。

等价地，$\boldsymbol{\pi}$ 是 $\mathbf{P}^T$ 的特征值 1 对应的左特征向量。

物理含义：若系统在时刻 $n$ 的分布为平稳分布 $\boldsymbol{\pi}$，则时刻 $n+1$ 的分布仍为 $\boldsymbol{\pi}$——分布"不再变化"。这并不意味着系统不运动，而是说从统计集合（ensemble）的角度看，各状态的占比保持恒定。

平稳分布的存在性与唯一性

定理 — 遍历链的基本极限定理

若 Markov 链是不可约、正常返且非周期的（即遍历链），则：

  1. 存在唯一的平稳分布 $\boldsymbol{\pi}$。
  2. 对所有 $i, j$：$\lim_{n \to \infty} P_{ij}^{(n)} = \pi_j$。
  3. $\pi_j = 1/\mu_j$，其中 $\mu_j$ 是状态 $j$ 的平均回返时间。
含义：在遍历链中，无论从哪个初始状态出发，经过足够长时间后，处于状态 $j$ 的概率趋近于 $\pi_j = 1/\mu_j$。这建立了"时间平均"与"集合平均"之间的联系——系统在状态 $j$ 逗留的长期时间比例等于 $\pi_j$。

详细平衡条件 (Detailed Balance)

定义 — 详细平衡 (Detailed Balance / Reversibility)

若概率向量 $\boldsymbol{\pi}$ 满足详细平衡方程：

$$\pi_i P_{ij} = \pi_j P_{ji} \quad \text{对所有 } i, j \in \mathcal{S}$$

则 $\boldsymbol{\pi}$ 是平稳分布。满足详细平衡的 Markov 链称为可逆的（reversible）。

验证

若详细平衡成立，对 $i$ 求和：

$$\sum_i \pi_i P_{ij} = \sum_i \pi_j P_{ji} = \pi_j \sum_i P_{ji} = \pi_j \cdot 1 = \pi_j$$

这正是 $\boldsymbol{\pi} = \boldsymbol{\pi}\mathbf{P}$ 的分量形式。$\square$

生灭链的平稳分布

定理 — 生灭链的平稳分布

不可约正常返生灭链满足详细平衡。其平稳分布为：

$$\pi_n = \pi_0 \cdot \frac{p_0 p_1 \cdots p_{n-1}}{q_1 q_2 \cdots q_n} = \pi_0 \cdot \gamma_n, \quad n \geq 1$$

归一化常数为 $\pi_0 = \left(1 + \sum_{n=1}^{\infty} \gamma_n\right)^{-1}$。平稳分布存在（有限）当且仅当 $\sum_{n=1}^{\infty} \gamma_n < \infty$。

推导

生灭链的详细平衡方程 $\pi_n p_n = \pi_{n+1} q_{n+1}$ 给出递推：

$$\pi_{n+1} = \frac{p_n}{q_{n+1}} \pi_n$$

逐步展开：$\pi_n = \frac{p_0 p_1 \cdots p_{n-1}}{q_1 q_2 \cdots q_n} \pi_0 = \gamma_n \pi_0$。归一化条件 $\sum_{n=0}^{\infty} \pi_n = 1$ 给出 $\pi_0$。$\square$

M/M/1 队列的平稳分布

对 M/M/1 队列，$\gamma_n = \rho^n$（其中 $\rho = \lambda/\mu$）。当 $\rho < 1$ 时：

$$\pi_0 = \left(1 + \sum_{n=1}^{\infty} \rho^n\right)^{-1} = \left(\frac{1}{1-\rho}\right)^{-1} = 1 - \rho$$ $$\pi_n = (1 - \rho)\rho^n, \quad n = 0, 1, 2, \ldots$$

这是几何分布。队列的平均长度为：

$$E[Q] = \sum_{n=0}^{\infty} n\pi_n = \frac{\rho}{1-\rho}$$
Little 定律验证：由 Little 定律 $L = \lambda W$，平均系统中顾客数 $L = \rho/(1-\rho)$，平均逗留时间 $W = L/\lambda = 1/(\mu - \lambda)$。当 $\rho \to 1$ 时（系统接近满载），队列长度和等待时间趋于无穷——这就是排队论中著名的"拥塞现象"。

10. 从 Poisson 到 Markov：概念桥梁

本讲的核心线索是从 Poisson 过程到 Markov 链的概念过渡。让我们梳理这一桥梁：

Poisson 过程Markov 链
连续时间，连续状态空间的计数离散时间（或连续时间），可数状态空间
无记忆性来自指数分布Markov 性质（给定现在，未来与过去独立）
平稳独立增量时齐转移概率
到达的"小增量"描述一步转移概率 $P_{ij}$
$N(t) \sim \text{Pois}(\lambda t)$$n$ 步分布 $\boldsymbol{\pi}^{(n)} = \boldsymbol{\pi}^{(0)}\mathbf{P}^n$
到达速率 $\lambda$平稳分布 $\pi_j = 1/\mu_j$
统一视角：Poisson 过程可以被视为一个特殊的连续时间 Markov 链，其状态空间为 $\{0, 1, 2, \ldots\}$，从状态 $n$ 以速率 $\lambda$ 跳到状态 $n+1$——这正是一个纯生过程（pure birth process）。将"生"和"死"（到达和离去）结合起来就得到生灭过程（如 M/M/1 队列）。因此 Markov 链是 Poisson 过程思想的自然推广。
