MIT 6.262 离散随机过程

第二讲：期望、独立性与 Bernoulli 过程

Lecturer: Prof. Robert Gallager (YouTube)

1. 期望 (Expectation)

随机变量 $X$ 的分布函数通常包含比实际需要更多的细节。期望 $\bar{X} = E[X]$ 是对随机变量最简洁的单一数值概括，它提供了"平均值"的精确数学含义。

定义 — 期望 (Expectation)

离散情形：若 $X$ 是离散随机变量，取值为 $a_1, a_2, \ldots$，则

$$E[X] = \sum_i a_i \, p_X(a_i)$$

连续情形：若 $X$ 具有概率密度函数 $f_X(x)$，则

$$E[X] = \int_{-\infty}^{\infty} x \, f_X(x) \, dx$$

上述两种定义要求级数或积分绝对收敛，否则期望不存在。

定义 — 互补 CDF 表示（非负随机变量）

对于任意非负随机变量 $X \ge 0$（不要求离散或连续），期望等于互补分布函数下的面积：

$$E[X] = \int_0^{\infty} F_X^c(x) \, dx, \qquad F_X^c(x) = \Pr\{X > x\} = 1 - F_X(x)$$

几何解释：考虑离散情形，$X$ 取正值 $a_1 < a_2 < a_3 < \cdots$。互补 CDF $F_X^c(x)$ 是一个递减阶梯函数。将 $\int_0^\infty F_X^c(x)\,dx$ 理解为阶梯下方的面积，可以按水平切片重排：在高度 $a_i$ 处的水平条对应宽度为 $(a_i - a_{i-1})$、高度为 $\Pr\{X \ge a_i\}$ 的矩形。将所有水平条累加恰好等于 $\sum_i a_i \, p_X(a_i) = E[X]$。对连续情形，同样的论证可通过对 Riemann 和取极限推广。

一般随机变量的互补 CDF 表示

对于可正可负的一般随机变量 $X$，将其分解为正部与负部后可得：

$$E[X] = -\int_{-\infty}^{0} F_X(x) \, dx + \int_0^{\infty} F_X^c(x) \, dx$$

其中第一项的几何含义是：对于 $x < 0$，$F_X(x) = \Pr\{X \le x\}$ 是 $X$ 落在负半轴某处的概率，对其在 $(-\infty, 0)$ 上积分给出负部的贡献（带负号）。这一公式比直接按密度积分在很多问题中更加方便。

实用提示：在许多排队论和可靠性理论问题中，随机变量天然非负（如服务时间、寿命），此时 $E[X] = \int_0^\infty F_X^c(x)\,dx$ 是最简洁的期望计算公式。它也是证明大量不等式（如 Markov 不等式）的起点。
定义 — 标准差 (Standard Deviation)

随机变量 $X$ 的标准差定义为方差的平方根：

$$\sigma_X = \sqrt{E\!\left[(X - \bar{X})^2\right]}$$

其中 $\bar{X} = E[X]$。标准差与 $X$ 量纲相同，衡量 $X$ 相对于其均值的"典型偏离程度"。方差 $\sigma_X^2 = E[(X-\bar{X})^2]$ 常更便于计算。

2. 指示随机变量 (Indicator Random Variables)

指示随机变量是连接事件与随机变量理论的桥梁。通过它，关于随机变量的一切定理都可以直接应用到事件上。

定义 — 指示随机变量 (Indicator RV)

对于概率模型中的事件 $A$，定义指示随机变量 $I_A$ 为：

$$I_A(\omega) = \begin{cases} 1, & \omega \in A \\ 0, & \omega \notin A \end{cases}$$

$I_A$ 是一个二值（binary）随机变量，其 PMF 为：

$$p_{I_A}(1) = \Pr\{A\}, \qquad p_{I_A}(0) = 1 - \Pr\{A\}$$

期望与标准差

直接由 PMF 计算：

$$E[I_A] = 1 \cdot \Pr\{A\} + 0 \cdot (1 - \Pr\{A\}) = \Pr\{A\}$$ $$E[I_A^2] = \Pr\{A\}, \qquad \text{Var}(I_A) = \Pr\{A\}(1 - \Pr\{A\})$$ $$\sigma_{I_A} = \sqrt{\Pr\{A\}\,(1 - \Pr\{A\})}$$

注意当 $\Pr\{A\} = 1/2$ 时标准差最大（$= 1/2$），当 $\Pr\{A\}$ 趋近 0 或 1 时标准差趋近 0 — 这符合直觉：越确定的事件，波动越小。

核心观察：关于随机变量的定理可以直接通过指示随机变量应用到事件上。例如，大数定律应用到 $I_A$ 就得到频率的稳定性；中心极限定理应用到 $I_A$ 就得到二项分布的正态近似。这一视角贯穿整个课程。

3. 多个随机变量与独立性

单个随机变量的分布函数 $F_X(x)$ 并不能完全刻画该随机变量与其他随机变量之间的关系。为此需要引入联合分布的概念。

定义 — 联合分布函数 (Joint CDF)

两个随机变量 $X, Y$ 的联合分布函数为：

$$F_{XY}(x, y) = \Pr\{X \le x,\; Y \le y\} = \Pr\!\big(\{\omega : X(\omega) \le x\} \cap \{\omega : Y(\omega) \le y\}\big)$$

更一般地，$n$ 个随机变量 $X_1, \ldots, X_n$ 的联合 CDF 为：

$$F_{\mathbf{X}}(x_1, \ldots, x_n) = \Pr\{X_1 \le x_1, \ldots, X_n \le x_n\}$$
定义 — 独立性 (Independence)

随机变量 $X_1, \ldots, X_n$ 相互独立当且仅当对所有 $x_1, \ldots, x_n \in \mathbb{R}$：

$$F_{\mathbf{X}}(x_1, \ldots, x_n) = \prod_{m=1}^n F_{X_m}(x_m)$$

即联合 CDF 等于各边际 CDF 的乘积。这一乘积形式同样适用于联合 PMF 和联合 PDF。

条件 PMF 与独立性的等价刻画

对于离散随机变量，条件 PMF 定义为：

$$p_{X|Y}(x|y) = \frac{p_{XY}(x, y)}{p_Y(y)}, \qquad \text{（当 } p_Y(y) > 0 \text{）}$$

利用条件 PMF，独立性有一个更直觉的等价表述：

$$X \text{ 与 } Y \text{ 独立} \iff p_{X|Y}(x|y) = p_X(x), \quad \forall\, x, y$$

即"知道 $Y$ 的值不改变 $X$ 的分布"。对密度函数类似：$f_{X|Y}(x|y) = f_X(x)$。

细节注意：若 $X_1, \ldots, X_n$ 相互独立，则它们的任意子集也相互独立。这对独立事件并不总是成立（两两独立不蕴含相互独立），但对独立随机变量是自动成立的，因为边际化联合 CDF 的乘积仍保持乘积结构。

4. IID 随机变量与扩展模型

定义 — IID (Independent and Identically Distributed)

随机变量 $X_1, X_2, \ldots, X_n$ 是独立同分布 (IID) 的，如果对所有 $x_1, \ldots, x_n$：

$$F_{\mathbf{X}}(x_1, \ldots, x_n) = \prod_{k=1}^n F_X(x_k)$$

即它们相互独立，且每个 $X_k$ 都具有相同的边际分布 $F_X$。该乘积形式同样适用于 PMF 和 PDF。

扩展模型的构造

给定一个概率模型（样本空间 $\mathbb{R}$，随机变量 $X$），我们总可以构造一个扩展模型：以 $\mathbb{R}^n$ 为样本空间，其上定义 IID 随机变量 $X_1, X_2, \ldots, X_n$。进一步令 $n \to \infty$，得到 IID 随机过程 $X_1, X_2, \ldots$。这对应于现实中"重复独立实验"的数学抽象。

样本均值与大数定律

定义部分和 $S_n = X_1 + X_2 + \cdots + X_n$，样本均值为 $S_n / n$。

大数定律的直觉：$S_n/n$ 随着 $n$ 增大"本质上变为确定性的"（essentially becomes deterministic），趋近于 $\bar{X} = E[X]$。

定量分析方差的行为：

  • $\sigma^2_{S_n} = n\sigma^2$：部分和的方差线性增长，分布的"宽度" $\sigma_{S_n} = \sigma\sqrt{n}$ 按 $\sqrt{n}$ 增长。
  • $\sigma^2_{S_n/n} = \sigma^2 / n$：样本均值的方差按 $1/n$ 缩小，分布的宽度按 $1/\sqrt{n}$ 缩小。

因此样本均值的分布越来越集中于 $\bar{X}$ 附近。当 $X$ 是事件 $A$ 的指示变量 $I_A$ 时，$S_n/n$ 即为事件 $A$ 的相对频率，大数定律就是"频率趋近概率"的严格表述。

模型的两类困难：(1) 现实中的重复实验不够"相似且独立"，不能用 IID 扩展模型来描述；(2) IID 扩展本身可以，但基本模型选取不当。理解模型局限性是应用概率论时的核心判断力。

5. Bernoulli 过程

定义 — Bernoulli 过程

Bernoulli 过程是一个 IID 二值随机变量序列 $Y_1, Y_2, Y_3, \ldots$，其中每个 $Y_k$ 的 PMF 为：

$$p_Y(1) = p > 0, \qquad p_Y(0) = 1 - p = q > 0$$

$Y_k = 1$ 通常称为"成功"（success），$Y_k = 0$ 称为"失败"（failure）。参数 $p$ 称为成功概率。

$S_n$ 的二项分布

令 $S_n = Y_1 + Y_2 + \cdots + Y_n$ 为前 $n$ 次试验中的成功总数。则 $S_n$ 服从参数 $(n, p)$ 的二项分布：

$$p_{S_n}(k) = \binom{n}{k} p^k q^{n-k}, \qquad k = 0, 1, \ldots, n$$

推导：任何具有 $k$ 个 1 和 $n-k$ 个 0 的特定 $n$-tuple 的概率为 $p^k q^{n-k}$（由 IID 性）。这样的 $n$-tuple 共有 $\binom{n}{k}$ 个（选择 $k$ 个位置放 1），因此 $p_{S_n}(k) = \binom{n}{k} p^k q^{n-k}$。

定理 — 二项分布的比率分析

相邻概率值的比为：

$$\frac{p_{S_n}(k+1)}{p_{S_n}(k)} = \frac{(n-k)\,p}{(k+1)\,q}$$

该比率关于 $k$ 严格递减。具体地：

  • 当 $k \ge pn$ 时，比率 $< 1$，即 $p_{S_n}(k)$ 递减
  • 当 $k + 1 \le pn$ 时，比率 $> 1$，即 $p_{S_n}(k)$ 递增
  • 当 $k < pn < k+1$ 时，比率 $\approx 1$，即 $p_{S_n}(k)$ 取峰值

因此 PMF 的峰值（mode）位于 $k \approx pn$ 附近，恰等于期望 $E[S_n] = np$。

推导：

$$\frac{p_{S_n}(k+1)}{p_{S_n}(k)} = \frac{\binom{n}{k+1} p^{k+1} q^{n-k-1}}{\binom{n}{k} p^k q^{n-k}} = \frac{n!/(k+1)!(n-k-1)!}{n!/k!(n-k)!} \cdot \frac{p}{q} = \frac{n-k}{k+1} \cdot \frac{p}{q}$$

令比率等于 1：$(n-k)p = (k+1)q$，解得 $k = pn - q$，即模在 $\lfloor pn \rfloor$ 或 $\lceil pn \rceil - 1$ 处取到。

6. Bernoulli 过程的中心极限定理推导

本节利用比率公式，通过对数近似直接推导 Bernoulli 和 $S_n$ 的 PMF 在峰值附近的 Gaussian 形状。这是中心极限定理的一个优美的初等证明。

Step 1：在 $k = pn$ 附近取对数

为简化代数，假设 $pn$ 为整数。设 $k = pn + i$（$i$ 相对较小），代入比率公式：

$$\frac{p_{S_n}(pn + i + 1)}{p_{S_n}(pn + i)} = \frac{n - pn - i}{pn + i + 1} \cdot \frac{p}{q} = \frac{nq - i}{pn + i + 1} \cdot \frac{p}{q}$$

改写为更便于取对数的形式：

$$\frac{p_{S_n}(pn + i + 1)}{p_{S_n}(pn + i)} = \frac{1 - \dfrac{i}{nq}}{1 + \dfrac{i+1}{np}}$$

取对数：

$$\ln\!\left[\frac{p_{S_n}(pn + i + 1)}{p_{S_n}(pn + i)}\right] = \ln\!\left(1 - \frac{i}{nq}\right) - \ln\!\left(1 + \frac{i+1}{np}\right)$$

Step 2：Taylor 展开

当 $|i| \ll n$ 时，利用 $\ln(1+x) \approx x - x^2/2 + \cdots$：

$$\ln\!\left(1 - \frac{i}{nq}\right) \approx -\frac{i}{nq} - \frac{i^2}{2n^2 q^2} - \cdots$$ $$\ln\!\left(1 + \frac{i+1}{np}\right) \approx \frac{i+1}{np} - \frac{(i+1)^2}{2n^2 p^2} - \cdots$$

相减，忽略 $O(i^2/n^2)$ 项：

$$\ln\!\left[\frac{p_{S_n}(pn + i + 1)}{p_{S_n}(pn + i)}\right] \approx -\frac{i}{nq} - \frac{i+1}{np} + \cdots = -\frac{i}{npq} - \frac{1}{np} + \cdots$$

其中利用了 $\frac{1}{nq} + \frac{1}{np} = \frac{p+q}{npq} = \frac{1}{npq}$。高阶修正项为 $O(i^2/n^2)$。

关键观察：对数增量近似地线性依赖于 $i$，系数为 $-1/(npq)$。这意味着 PMF 的对数近似为 $i$ 的二次函数 — 即 Gaussian 形状。

Step 3：伸缩求和 (Telescoping Sum)

将从 $pn$ 到 $pn + j$ 的变化表示为 $j$ 个单位增量的和：

$$\ln\!\left[\frac{p_{S_n}(pn + j)}{p_{S_n}(pn)}\right] = \sum_{i=0}^{j-1} \ln\!\left[\frac{p_{S_n}(pn + i + 1)}{p_{S_n}(pn + i)}\right]$$

代入近似式：

$$\approx \sum_{i=0}^{j-1}\left(-\frac{i}{npq} - \frac{1}{np} + \cdots\right) = -\frac{1}{npq}\sum_{i=0}^{j-1} i - \frac{j}{np} + \cdots$$ $$= -\frac{j(j-1)}{2npq} - \frac{j}{np} + \cdots \approx -\frac{j^2}{2npq}$$

其中利用了 $\sum_{i=0}^{j-1} i = j(j-1)/2 \approx j^2/2$，并忽略了与 $j^2$ 相比的线性项（线性项的量级等同于单位增量，远小于主项）。

Step 4：得出 Gaussian 形式

由伸缩和结果：

$$\ln\!\left[\frac{p_{S_n}(pn + j)}{p_{S_n}(pn)}\right] \approx -\frac{j^2}{2npq}$$

取指数：

$$p_{S_n}(pn + j) \approx p_{S_n}(pn) \cdot \exp\!\left(-\frac{j^2}{2npq}\right)$$

此公式对 $j$ 正负均成立，是一个以 $pn$ 为中心的量子化 Gaussian 分布，具有未知的归一化常数 $p_{S_n}(pn)$。

定理 — Bernoulli 和的中心极限定理（离散 PMF 形式）

选取归一化常数使得 PMF 求和为 1（利用 Gaussian 积分 $\int_{-\infty}^\infty e^{-t^2/2}\,dt = \sqrt{2\pi}$），得到：

$$p_{S_n}(pn + j) \approx \frac{1}{\sqrt{2\pi npq}} \exp\!\left(-\frac{j^2}{2npq}\right)$$

等价地，标准化随机变量 $Z_n = \dfrac{S_n - np}{\sqrt{npq}}$ 的分布近似于标准正态分布：

$$\lim_{n\to\infty} \Pr\!\left\{\frac{S_n - np}{\sqrt{npq}} \le y\right\} = \int_{-\infty}^y \frac{1}{\sqrt{2\pi}} e^{-x^2/2}\,dx = \Phi(y)$$

即 $S_n$ 近似服从均值 $np$、方差 $npq$ 的正态分布 $\mathcal{N}(np,\, npq)$。

推导回顾：整个证明只依赖于：(1) 比率公式 $(n-k)p/((k+1)q)$；(2) 在峰值附近 Taylor 展开 $\ln(1+x)$；(3) 伸缩求和。无需 Stirling 公式或特征函数。教材 1.5.3 节给出了基于特征函数的另一种证明。

本讲小结

主题要点
期望离散：$E[X]=\sum a_i p_X(a_i)$；连续：$E[X]=\int xf_X(x)\,dx$；非负通用：$E[X]=\int_0^\infty F_X^c(x)\,dx$
标准差$\sigma_X = \sqrt{E[(X-\bar X)^2]}$，衡量随机变量的"典型偏离程度"
指示随机变量$I_A(\omega)=\mathbf{1}(\omega\in A)$；$E[I_A]=\Pr\{A\}$；事件定理可转化为 rv 定理
联合分布与独立性$X_1,\ldots,X_n$ 独立 $\iff$ $F_{\mathbf{X}}=\prod F_{X_m}$；条件 PMF 等于边际 PMF
IID 与样本均值$\sigma^2_{S_n/n}=\sigma^2/n$，分布宽度按 $1/\sqrt{n}$ 缩小，大数定律的定量基础
Bernoulli 过程IID 二值过程；$S_n\sim\text{Binomial}(n,p)$；比率 $(n-k)p/((k+1)q)$ 确定峰值在 $k\approx pn$
CLT 推导对数比率线性 $\to$ 伸缩和为二次 $\to$ $p_{S_n}(pn+j)\approx\frac{1}{\sqrt{2\pi npq}}e^{-j^2/(2npq)}$
