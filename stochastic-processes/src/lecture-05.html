第五讲：Poisson 过程的合并与分裂

Lecturer: Prof. Robert Gallager (YouTube)

1. Poisson 过程回顾

在开始讨论合并与分裂之前，我们先回顾 Poisson 过程的核心性质，它们将在本讲中反复使用。

速率为 $\lambda$ 的 Poisson 过程 $\{N(t);\, t > 0\}$ 具有以下等价刻画：

  • 到达间隔刻画：到达间隔 $X_1, X_2, \ldots$ 是 IID $\text{Exp}(\lambda)$ 随机变量。
  • 增量刻画：过程具有平稳独立增量，且 $N(t) \sim \text{Poisson}(\lambda t)$。
  • 小增量刻画：$\Pr\{\widetilde{N}(t, t+\delta) = 1\} = \lambda\delta + o(\delta)$，$\Pr\{\widetilde{N}(t, t+\delta) \geq 2\} = o(\delta)$。

关键性质：对任意时间划分 $0 < t_1 < t_2 < \cdots < t_k$，各区间上的增量 $\widetilde{N}(t_{i-1}, t_i) = N(t_i) - N(t_{i-1})$ 是互相独立的 Poisson 随机变量，参数分别为 $\lambda(t_i - t_{i-1})$。

独立 Poisson 随机变量的可加性：若 $Y_1 \sim \text{Poisson}(\mu_1)$，$Y_2 \sim \text{Poisson}(\mu_2)$ 独立，则 $Y_1 + Y_2 \sim \text{Poisson}(\mu_1 + \mu_2)$。这可由 MGF 直接验证：$E[e^{r(Y_1+Y_2)}] = e^{\mu_1(e^r-1)} \cdot e^{\mu_2(e^r-1)} = e^{(\mu_1+\mu_2)(e^r-1)}$。此性质是合并定理的代数基础。

2. 独立 Poisson 过程的合并 (Combining/Superposition)

定义 — 独立 Poisson 过程

两个 Poisson 过程 $\{N_1(t);\, t > 0\}$（速率 $\lambda_1$）和 $\{N_2(t);\, t > 0\}$（速率 $\lambda_2$）称为独立的，如果对任意时刻集 $t_1, \ldots, t_n$，随机向量 $(N_1(t_1), \ldots, N_1(t_n))$ 与 $(N_2(t_1), \ldots, N_2(t_n))$ 独立。

定理 — Poisson 过程的合并 (Superposition Theorem)

设 $\{N_1(t);\, t > 0\}$ 和 $\{N_2(t);\, t > 0\}$ 是速率分别为 $\lambda_1$ 和 $\lambda_2$ 的独立 Poisson 过程。定义合并过程：

$$N(t) = N_1(t) + N_2(t)$$

则 $\{N(t);\, t > 0\}$ 是速率为 $\lambda = \lambda_1 + \lambda_2$ 的 Poisson 过程。

证明方法一（小增量法）

在任意小增量 $(t, t+\delta]$ 内，$\widetilde{N}(t, t+\delta) = \widetilde{N}_1(t, t+\delta) + \widetilde{N}_2(t, t+\delta)$。

恰好一次到达的概率：

$$\Pr\{\widetilde{N}(t, t+\delta) = 1\} = \Pr\{\widetilde{N}_1 = 1\}\Pr\{\widetilde{N}_2 = 0\} + \Pr\{\widetilde{N}_1 = 0\}\Pr\{\widetilde{N}_2 = 1\}$$ $$= [\lambda_1\delta + o(\delta)][1 - \lambda_2\delta + o(\delta)] + [1 - \lambda_1\delta + o(\delta)][\lambda_2\delta + o(\delta)]$$ $$= (\lambda_1 + \lambda_2)\delta + o(\delta)$$

零次到达的概率：

$$\Pr\{\widetilde{N}(t, t+\delta) = 0\} = [1 - \lambda_1\delta + o(\delta)][1 - \lambda_2\delta + o(\delta)] = 1 - (\lambda_1+\lambda_2)\delta + o(\delta)$$

两次及以上到达的概率：由补概率，$\Pr\{\widetilde{N} \geq 2\} = o(\delta)$。

合并过程继承了平稳独立增量（由两个过程的独立性和各自的平稳独立增量保证），且满足小增量条件（参数 $\lambda_1 + \lambda_2$）。由等价定义 II，$\{N(t)\}$ 是速率 $\lambda_1 + \lambda_2$ 的 Poisson 过程。$\square$

证明方法二（直接用 Poisson 可加性）

对任意不相交区间 $(t_{i-1}, t_i]$，$\widetilde{N}_1(t_{i-1}, t_i) \sim \text{Pois}(\lambda_1(t_i - t_{i-1}))$ 与 $\widetilde{N}_2(t_{i-1}, t_i) \sim \text{Pois}(\lambda_2(t_i - t_{i-1}))$ 独立。由 Poisson 可加性：

$$\widetilde{N}(t_{i-1}, t_i) = \widetilde{N}_1(t_{i-1}, t_i) + \widetilde{N}_2(t_{i-1}, t_i) \sim \text{Pois}((\lambda_1+\lambda_2)(t_i - t_{i-1}))$$

且不同区间上的增量仍独立（因为 $N_1, N_2$ 各自独立增量且彼此独立）。$\square$

推广到多个过程

更一般地，若 $\{N_1(t)\}, \{N_2(t)\}, \ldots, \{N_k(t)\}$ 是互相独立的 Poisson 过程，速率分别为 $\lambda_1, \ldots, \lambda_k$，则合并过程 $N(t) = \sum_{i=1}^k N_i(t)$ 是速率为 $\lambda = \sum_{i=1}^k \lambda_i$ 的 Poisson 过程。

大量小过程的近似：即使各个子过程不是 Poisson 的，只要它们大量存在、互相独立、且每个过程的贡献很小，合并后的过程仍然近似 Poisson。这类似于 CLT 的精神——独立性在叠加中"洗掉"了各子过程内部的依赖结构。这解释了为什么 Poisson 过程在实际中如此普遍。

3. Poisson 过程的分裂/稀释 (Splitting/Thinning)

定理 — Poisson 过程的分裂 (Splitting/Thinning Theorem)

设 $\{N(t);\, t > 0\}$ 是速率为 $\lambda$ 的 Poisson 过程。每次到达独立地以概率 $p$ 被分配到"类型 1"，以概率 $1-p$ 分配到"类型 2"（分配决策由一个与 $N(t)$ 独立的 Bernoulli 过程控制）。定义：

  • $N_1(t)$ = 时间 $(0, t]$ 内类型 1 的到达数
  • $N_2(t)$ = 时间 $(0, t]$ 内类型 2 的到达数

则：

  1. $\{N_1(t);\, t > 0\}$ 是速率为 $\lambda p$ 的 Poisson 过程。
  2. $\{N_2(t);\, t > 0\}$ 是速率为 $\lambda(1-p)$ 的 Poisson 过程。
  3. $\{N_1(t)\}$ 和 $\{N_2(t)\}$ 是独立的。
证明（小增量法）

步骤 1：各子过程是 Poisson。

在小增量 $(t, t+\delta]$ 内，$\widetilde{N}_1(t, t+\delta)$ 的分布为：

$$\Pr\{\widetilde{N}_1 = 1\} = \Pr\{\widetilde{N} = 1\} \cdot p + o(\delta) = \lambda p \delta + o(\delta)$$ $$\Pr\{\widetilde{N}_1 = 0\} = \Pr\{\widetilde{N} = 0\} + \Pr\{\widetilde{N} = 1\}(1-p) + o(\delta) = 1 - \lambda p \delta + o(\delta)$$ $$\Pr\{\widetilde{N}_1 \geq 2\} = o(\delta)$$

$N_1$ 继承了原过程的平稳独立增量性质（因为分配决策在每个增量上独立进行）。由等价定义 II，$N_1$ 是速率 $\lambda p$ 的 Poisson 过程。类似地 $N_2$ 是速率 $\lambda(1-p)$ 的 Poisson 过程。

步骤 2：$N_1$ 和 $N_2$ 独立。

对任意区间 $(t, t+\delta]$，给定 $\widetilde{N}(t, t+\delta) = n$，$\widetilde{N}_1$ 和 $\widetilde{N}_2 = n - \widetilde{N}_1$ 的联合分布为 Binomial 分裂。由 Poisson 分裂引理（见下），$\widetilde{N}_1$ 和 $\widetilde{N}_2$ 无条件独立。对不同区间的独立性由原过程的独立增量保证。$\square$

引理 — Poisson 分裂引理

设 $N \sim \text{Poisson}(\mu)$。给定 $N = n$，每个事件独立地以概率 $p$ 归入类型 1。设 $N_1$ = 类型 1 的个数，$N_2 = N - N_1$。则 $N_1 \sim \text{Poisson}(\mu p)$，$N_2 \sim \text{Poisson}(\mu(1-p))$，且 $N_1$ 与 $N_2$ 独立。

证明（Poisson 分裂引理）

计算联合 PMF：

$$\Pr\{N_1 = j,\, N_2 = k\} = \Pr\{N_1 = j,\, N_2 = k \mid N = j+k\} \cdot \Pr\{N = j+k\}$$ $$= \binom{j+k}{j} p^j (1-p)^k \cdot \frac{\mu^{j+k} e^{-\mu}}{(j+k)!}$$ $$= \frac{p^j (1-p)^k}{j!\, k!} \cdot \mu^{j+k} e^{-\mu}$$ $$= \frac{(\mu p)^j e^{-\mu p}}{j!} \cdot \frac{(\mu(1-p))^k e^{-\mu(1-p)}}{k!}$$

最后一步利用了 $e^{-\mu} = e^{-\mu p} \cdot e^{-\mu(1-p)}$。联合 PMF 分解为两个独立 Poisson 的乘积。$\square$

直觉：分裂后的两个过程独立这一事实乍看反直觉——毕竟 $N_1(t) + N_2(t) = N(t)$，知道一个似乎应该限制另一个。但 Poisson 分布的特殊结构使得条件化后恰好"消除"了这种约束。这是 Poisson 分布的独特代数性质。

推广到多路分裂

更一般地，每次到达可以独立地分配到 $k$ 类之一，概率分别为 $p_1, \ldots, p_k$（$\sum p_i = 1$）。则各类形成的子过程 $\{N_i(t)\}$ 是互相独立的 Poisson 过程，速率分别为 $\lambda p_1, \ldots, \lambda p_k$。

4. 条件到达密度与顺序统计量

Poisson 过程的另一个深刻性质是：给定区间内的到达总数后，各到达时刻的条件分布是均匀的。这与顺序统计量（order statistics）之间存在精确对应。

给定 $N(t) = 1$ 时的条件分布

定理 — 单次到达的条件均匀性

给定 $N(t) = 1$（即 $(0, t]$ 内恰好一次到达），到达时刻 $S_1$ 的条件密度为：

$$f_{S_1 | N(t)}(s_1 \mid 1) = \frac{1}{t}, \quad 0 < s_1 < t$$

即 $S_1 \mid N(t)=1$ 在 $(0, t]$ 上均匀分布。

证明

利用独立增量。事件 "$S_1 \in (s_1, s_1+\delta]$ 且 $N(t) = 1$" 等价于"$(0, s_1]$ 无到达，$(s_1, s_1+\delta]$ 恰一次到达，$(s_1+\delta, t]$ 无到达"：

$$f_{S_1|N(t)}(s_1 \mid 1) = \lim_{\delta \to 0} \frac{\Pr\{N(s_1) = 0\} \cdot \Pr\{\widetilde{N}(s_1, s_1+\delta) = 1\} \cdot \Pr\{\widetilde{N}(s_1+\delta, t) = 0\}}{\delta \cdot p_{N(t)}(1)}$$ $$= \frac{e^{-\lambda s_1} \cdot \lambda\delta \cdot e^{-\lambda(t - s_1 - \delta)}}{\delta \cdot \lambda t e^{-\lambda t}} = \frac{e^{-\lambda s_1} \cdot \lambda \cdot e^{-\lambda(t-s_1)}}{{\lambda t e^{-\lambda t}}} = \frac{\lambda e^{-\lambda t}}{\lambda t e^{-\lambda t}} = \frac{1}{t} \qquad\square$$

给定 $N(t) = n$ 时的联合条件分布

定理 — 条件联合密度（核心定理）

给定 $N(t) = n$，到达时刻向量 $(S_1, \ldots, S_n)$ 的条件联合密度为：

$$f_{\vec{S}^{(n)} | N(t)}(\vec{s}^{(n)} \mid n) = \frac{n!}{t^n}, \quad 0 < s_1 < s_2 < \cdots < s_n < t$$

这是在有序区域 $\{0 < s_1 < \cdots < s_n < t\}$ 上的均匀分布。

证明

将 $(0, t]$ 在 $s_1, \ldots, s_n$ 处划分为 $n+1$ 个子区间。利用独立增量：

$$f_{\vec{S}^{(n)}|N(t)}(\vec{s}^{(n)}|n) = \lim_{\delta \to 0} \frac{\prod_{i=1}^n [e^{-\lambda(s_i - s_{i-1}-\delta)} \cdot \lambda\delta] \cdot e^{-\lambda(t-s_n-\delta)}}{\delta^n \cdot p_{N(t)}(n)}$$

其中 $s_0 = 0$。指数项合并：

$$\prod_{i=1}^n e^{-\lambda(s_i - s_{i-1} - \delta)} \cdot e^{-\lambda(t-s_n-\delta)} = e^{-\lambda(t - n\delta)} \to e^{-\lambda t}$$

因此：

$$f_{\vec{S}^{(n)}|N(t)}(\vec{s}^{(n)}|n) = \frac{(\lambda\delta)^n \cdot e^{-\lambda t}}{\delta^n \cdot \frac{(\lambda t)^n e^{-\lambda t}}{n!}} = \frac{\lambda^n \cdot e^{-\lambda t}}{\frac{(\lambda t)^n e^{-\lambda t}}{n!}} = \frac{n!}{t^n} \qquad\square$$
关键观察：条件密度 $n!/t^n$ 不依赖于 $\lambda$！给定到达总数后，速率参数的信息完全被"消耗"在确定 $n$ 的过程中；到达时刻的空间分布则完全由均匀性决定。这使得许多条件计算变得极其简洁。

顺序统计量的对应

定理 — 顺序统计量等价 (Order Statistics Equivalence)

设 $U_1, \ldots, U_n$ 是 $n$ 个独立的 $\text{Uniform}(0, t]$ 随机变量，$S_1 \leq S_2 \leq \cdots \leq S_n$ 是它们的顺序统计量（order statistics），即 $S_k$ 是第 $k$ 小的值。则：

$(S_1, \ldots, S_n)$ 的联合密度恰好等于 Poisson 过程中给定 $N(t) = n$ 时到达时刻的条件联合密度。

证明

$\vec{U}^n = (U_1, \ldots, U_n)$ 的联合密度在 $(0, t]^n$（体积 $t^n$）上均匀，为 $1/t^n$。

$n$ 维空间 $(0, t]^n$ 可按坐标排列方式划分为 $n!$ 个等体积区域（对应 $n!$ 种排列）。由对称性，每个区域的体积为 $t^n / n!$。

顺序统计量 $(S_1, \ldots, S_n)$ 的支撑集恰好是其中一个区域 $\{0 < s_1 < \cdots < s_n < t\}$，体积为 $t^n/n!$。在该区域上的密度为 $n!/t^n$（对原密度 $1/t^n$ 乘以 Jacobian $n!$，即从 $n!$ 个排列映射到同一组顺序统计量）。$\square$

实用意义：这个等价意味着我们可以用顺序统计量的已知结果来回答 Poisson 过程的条件问题，反之亦然。例如：
- 要模拟给定 $N(t) = n$ 的 Poisson 过程，只需生成 $n$ 个 $\text{Uniform}(0, t)$ 随机数然后排序。
- 顺序统计量的边缘分布、矩等标准结果可以直接搬用。

边缘条件分布的推导

利用顺序统计量等价，可以快速得到各种边缘分布：

$S_1 \mid N(t) = n$ 的分布：$S_1 = \min(U_1, \ldots, U_n)$，其互补 CDF 为：

$$\Pr\{S_1 > s_1 \mid N(t) = n\} = \Pr\{\min_i U_i > s_1\} = \prod_{i=1}^n \Pr\{U_i > s_1\} = \left(1 - \frac{s_1}{t}\right)^n$$

密度为：

$$f_{S_1|N(t)}(s_1 \mid n) = \frac{n}{t}\left(1 - \frac{s_1}{t}\right)^{n-1}, \quad 0 < s_1 < t$$

条件期望：

$$E[S_1 \mid N(t) = n] = \frac{t}{n+1}$$
对称性：由于 $n$ 个均匀随机变量将 $(0, t]$ 划分为 $n+1$ 段，由对称性，每段的期望长度为 $t/(n+1)$。因此 $E[S_k \mid N(t) = n] = kt/(n+1)$，对 $k = 1, \ldots, n$ 均成立。

条件下到达间隔的分布

给定 $N(t) = n$，定义 $X_k = S_k - S_{k-1}$（$k = 1, \ldots, n$）以及剩余时间 $X_{n+1}^* = t - S_n$。由从 $(s_1, \ldots, s_n)$ 到 $(x_1, \ldots, x_n)$ 的线性变换（Jacobian = 1）：

$$f_{X_1, \ldots, X_n, X_{n+1}^* \mid N(t)}(x_1, \ldots, x_n, x_{n+1}^* \mid n) = \frac{n!}{t^n}$$

约束为 $x_i > 0$（$i = 1, \ldots, n+1$），$\sum_{i=1}^{n+1} x_i = t$。这是在 $n$ 维单纯形上的均匀分布（Dirichlet 分布的特例）。

$X_1$ 的边缘条件密度（对 $N(t) = 2$ 的情形）：

$$f_{X_1|N(t)}(x_1 \mid 2) = \frac{2}{t}\left(1 - \frac{x_1}{t}\right), \quad 0 < x_1 < t$$

一般情形：

$$f_{X_1|N(t)}(x_1 \mid n) = \frac{n}{t}\left(1 - \frac{x_1}{t}\right)^{n-1}, \quad 0 < x_1 < t$$

由对称性，$X_{n+1}^* = t - S_n$ 具有相同的条件分布：

$$f_{X_{n+1}^*|N(t)}(x \mid n) = \frac{n}{t}\left(1 - \frac{x}{t}\right)^{n-1}, \quad E[X_{n+1}^* \mid N(t) = n] = \frac{t}{n+1}$$

检验悖论 (Inspection Paradox) 的初步讨论

考虑无条件的期望 $E[X_{n+1}^*]$：

$$E[X_{N(t)+1}^*] = \sum_{n=0}^{\infty} \frac{t}{n+1} \cdot \frac{(\lambda t)^n e^{-\lambda t}}{n!} = \sum_{n=0}^{\infty} \frac{(\lambda t)^{n+1} e^{-\lambda t}}{\lambda(n+1)!} = \frac{1 - e^{-\lambda t}}{\lambda}$$

当 $t$ 很大时，$E[X_{N(t)+1}^*] \approx 1/\lambda$。而 $X_{N(t)+1}$（包含时刻 $t$ 的完整到达间隔）的期望为：

$$E[X_{N(t)+1}] = E[X_{N(t)+1}^*] + E[t - S_{N(t)}] \approx \frac{2}{\lambda} - e^{-\lambda t}/\lambda$$
检验悖论：包含随机选取时刻 $t$ 的到达间隔的期望长度约为 $2/\lambda$，是普通到达间隔期望 $1/\lambda$ 的两倍。直觉解释：如果你在一个随机时刻"跳入"过程，你更可能落在一个较长的到达间隔内（长区间占据更多时间），从而导致采样偏差。这一现象将在更新理论中深入讨论。

5. 非齐次 Poisson 过程 (Non-homogeneous Poisson Processes)

在许多实际场景中（如光通信中光子流的调制、一天中不同时段的电话到达），到达率随时间变化。这引出了非齐次（non-homogeneous）Poisson 过程。

定义 — 非齐次 Poisson 过程

一个计数过程 $\{N(t);\, t > 0\}$ 是非齐次 Poisson 过程，如果它满足：

  1. 独立增量：不相交区间上的增量相互独立。
  2. 时变小增量条件：对所有 $t > 0$，
$$\Pr\{\widetilde{N}(t, t+\delta) = n\} = \begin{cases} 1 - \lambda(t)\delta + o(\delta) & n = 0 \\ \lambda(t)\delta + o(\delta) & n = 1 \\ o(\delta) & n \geq 2 \end{cases}$$

其中 $\lambda(t) \geq 0$ 是速率函数（rate function / intensity function）。

定理 — 非齐次 Poisson 过程的增量分布

对非齐次 Poisson 过程，区间 $(t, \tau]$ 上的到达数服从 Poisson 分布：

$$\Pr\{\widetilde{N}(t, \tau) = n\} = \frac{[\widetilde{m}(t, \tau)]^n \exp[-\widetilde{m}(t, \tau)]}{n!}$$

其中累积强度（mean function）为：

$$\widetilde{m}(t, \tau) = \int_t^{\tau} \lambda(u)\, du$$
证明思路

将 $(t, \tau]$ 划分为 $m$ 个小段，每段长 $\delta = (\tau-t)/m$。第 $j$ 段上到达概率约为 $\lambda(t+j\delta)\delta$。当 $m \to \infty$ 时，这是 $m$ 次独立 Bernoulli 试验（成功概率之和趋向于 $\int_t^\tau \lambda(u)\,du$），由 Poisson 极限定理得结论。$\square$

与齐次情形的区别：
- 非齐次 Poisson 过程有独立增量但不再有平稳增量（增量分布取决于区间位置而非仅其长度）。
- 到达间隔不再是 IID 的，也不再是指数分布。
- 合并和分裂的性质仍然成立：两个独立的非齐次 Poisson 过程合并后仍是非齐次 Poisson（速率相加）；分裂也保持非齐次 Poisson 结构（速率按比例分配）。

6. 应用举例

例 1：后到先服务 (LCFS) 队列中的完成概率

模型：顾客按速率 $\lambda$ 的 Poisson 过程到达。服务时间为独立的 $\text{Exp}(\mu)$。采用后到先服务策略：新到达的顾客立刻开始服务，当前被服务的顾客被中断（preempted）。

问题：一个新到达的顾客能在被中断前完成服务的概率是多少？

解答：将服务完成也视为一个独立的 Poisson 过程（速率 $\mu$）。"下一个事件"是到达还是服务完成？由于到达过程（速率 $\lambda$）和服务过程（速率 $\mu$）是独立的 Poisson 过程，合并后速率为 $\lambda + \mu$，且下一个事件是服务完成的概率为：

$$\Pr\{\text{服务完成先于下一次到达}\} = \frac{\mu}{\lambda + \mu}$$

例 2：被中断后最终完成的概率

问题：给定你被中断了一次，你最终（在没有更多中断的情况下）完成服务的概率是多少？

解答：需要中断你的那位顾客先完成（概率 $\mu/(\lambda+\mu)$），然后你也需要在下一次到达前完成（概率 $\mu/(\lambda+\mu)$）。因此：

$$\Pr\{\text{被中断一次后完成}\} = \left(\frac{\mu}{\lambda + \mu}\right)^2$$

更一般地，被中断 $k$ 次后最终完成的概率为 $(\mu/(\lambda+\mu))^{k+1}$。

例 3：分裂的应用 — 网络路由

模型：数据包按速率 $\lambda$ 的 Poisson 过程到达路由器。路由器将每个包独立地以概率 $p_i$ 转发到第 $i$ 条链路（$i = 1, \ldots, k$，$\sum p_i = 1$）。

结论：第 $i$ 条链路上的数据包到达过程是速率为 $\lambda p_i$ 的 Poisson 过程，且各链路的到达过程互相独立。这大大简化了网络性能分析。

例 4：条件均匀性的应用

问题：一天中 $(0, T]$ 时间内有 $N(T) = 100$ 辆车通过检查站。估计第一辆车到达的期望时刻。

解答：由条件均匀性，$E[S_1 \mid N(T) = 100] = T/101$。更一般地，第 $k$ 辆车的期望到达时刻为 $kT/101$——100 辆车将 $(0, T]$ 均匀划分为 101 段。

7. 合并与分裂的对偶性总结

合并和分裂互为"逆运算"：

操作输入输出
合并 独立 Poisson 过程，速率 $\lambda_1, \ldots, \lambda_k$ Poisson 过程，速率 $\sum \lambda_i$
分裂 Poisson 过程速率 $\lambda$，分配概率 $p_1, \ldots, p_k$ 独立 Poisson 过程，速率 $\lambda p_1, \ldots, \lambda p_k$

若先将速率 $\lambda$ 的过程按 $(p_1, p_2)$ 分裂，再合并，回到原速率 $\lambda p_1 + \lambda p_2 = \lambda$。反之，合并后再分裂回去，每个子过程的速率也恢复原值。但注意：分裂再合并得到的过程与原过程分布相同（都是同速率 Poisson），但不是同一组随机变量（到达时刻不同）。

本讲小结

概念/结果内容要点
合并定理 $N_1(\lambda_1) + N_2(\lambda_2) \to N(\lambda_1+\lambda_2)$ 独立 Poisson 合并仍 Poisson；速率相加
分裂定理 $N(\lambda) \xrightarrow{p} N_1(\lambda p),\; N_2(\lambda(1-p))$ 子过程独立且仍 Poisson；速率按比例
Poisson 分裂引理 Pois$(\mu)$ 按 $p$ 分裂 $\to$ 独立 Pois$(\mu p)$, Pois$(\mu(1-p))$ 独立性是 Poisson 的代数特性
条件均匀性 $f_{\vec{S}^n|N(t)}= n!/t^n$（有序区域上均匀） 不依赖 $\lambda$；等价于顺序统计量
顺序统计量等价 $n$ 个 Unif$(0,t)$ 的顺序统计量 $\overset{d}{=}$ 条件到达时刻 可用于模拟和分布推导
非齐次 Poisson 速率 $\lambda(t)$ 随时间变化；$\widetilde{N}(t,\tau) \sim$ Pois$(\int_t^\tau \lambda(u)du)$ 独立增量但无平稳增量
