### 涵盖内容

- 赋范空间、完备性、Banach 空间
- 泛函、Hahn-Banach 定理、对偶性
- Lebesgue 测度、可测函数、L^p 空间
- Hilbert 空间、标准正交基、Fourier 级数
- 紧算子、自伴算子、谱定理

## 笔记制作方式

每讲笔记基于以下材料整理：

1. MIT OpenCourseWare 发布的**官方讲义**
2. 课程录像的**视频字幕**
3. **个人重组** — 以定义、定理、证明、小结的结构重新编排，中文撰写

数学公式使用 MathJax 排版，渲染为 PDF。
